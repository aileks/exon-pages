## Running Locally

1. Clone this repository
```bash
git clone https://github.com/aileks/exon-pages.git
cd exon-pages
```

2. Install the required dependencies
```bash
pipenv install -r requirements.txt
cd frontend
pnpm install
```

3. Run the application

```bash
pipenv run flask run
```
